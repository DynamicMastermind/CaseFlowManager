CAFO2017!

This is a field-work application for social workers. At this point, if you were using the application to perform actual field-work you would be able to interact with your caseload via mobile-friendly pages. For the purpose of the CAFO2017 conference, this site will allow you to:

1. Build a 'case' using your own contact information. We will use your contact information to follow-up with you at a later time. Only you will be able to view your 'case' data.
2. Create your own user account to access your contact information.
3. Create additional test cases that only you can view.
